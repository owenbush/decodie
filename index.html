Decodie

Your AI learning companion. Turn every coding session into a structured learning trail.

Get Started View on GitHub

How it works

You code

Write code with Claude Code as usual

Decodie observes

Documents decisions, patterns, and concepts

You learn

Follow lessons, track progress, and ask questions

See it in action

Lessons view — topic-based lessons with progress tracking
Lesson detail — ordered entries with sidebar outline
Entry detail — code snippets, explanations, and lesson navigation
Entries view — filter by level, topic, type, and lifecycle
Interactive Q&A — ask follow-up questions about any entry

Features

Automatic documentation

Learning entries are generated as the AI writes code. No extra steps needed. Use /decodie to activate.

Structured lessons

Entries are automatically grouped into topic-based lessons with sequential navigation. Create custom lessons with your own ordering and notes.

Progress tracking

Mark entries as learned and track your progress through each lesson with visual progress indicators.

Interactive Q&A

Highlight any text in an entry and ask follow-up questions. Conversations persist across sessions.

Content-based code references

References use function signatures, not line numbers, so they survive refactoring.

Smart filtering

Filter by experience level, topic, decision type, and lifecycle stage. Switch between lesson and entry views.

Analyze existing code

Run /decodie-analyze to generate entries from code you didn't write with Claude. Works on files, directories, or whole projects.

External doc links

Automatic links to php.net, MDN, Python docs, Drupal API, and framework docs.

DDEV integration

Serve the UI as a DDEV service at decodie.yoursite.ddev.site.

Repositories

decodie-skill

Claude Code skills for generating and analyzing .decodie/ entries.

decodie-ui

Node.js presentation layer with lessons, progress tracking, browsing, filtering, and Q&A.

decodie-ddev

DDEV add-on that serves the UI at decodie.SITENAME.ddev.site.

Quick Start

# 1. Install the skill + UI
npx @owenbush/decodie-ui install-skill

# 2. Activate the skill in Claude Code
/decodie

# 3. Code as usual — entries appear in .decodie/
#    Or analyze existing code:
/decodie-analyze src/

# 4. Browse your entries
npx @owenbush/decodie-ui serve

# -- or with DDEV (installs everything automatically) --
ddev add-on get owenbush/decodie-ddev
ddev restart
ddev decodie
